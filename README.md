# Student Scores Manager

A simple command-line (CLI) application written in Python to manage students and their scores.

## Features

- Add, delete, and search students
- Manage subject scores (add, update, delete)
- Display student information
- Calculate average scores (per student and all students)
- Save and load data using JSON

## How to Run

1. Make sure Python is installed
2. Run the program:
